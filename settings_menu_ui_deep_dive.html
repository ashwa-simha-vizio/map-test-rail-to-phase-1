Automation Recommendations (Priority Order)

1. Closed Captions (16 TCs, 100% gap)

Add a TC-CC-001 automation TC covering CC toggle, analog/digital modes, global setting behavior, and cross-input persistence. High regulatory/accessibility compliance risk.

2. Accessibility - TalkBack & Zoom (10 TCs, 100% gap)

Add TC-ACC-001/002 covering TalkBack enable/disable, language restrictions, and Zoom navigation mode activation. ADA/Section 508 compliance.

3. Quick Settings Menu Behavior (17 TCs, 82% gap)

Add TC-QS-001 covering Quick Settings dismissal paths (6 different button interactions), timeout behavior, and state management during content playback.

4. Network Settings UI Validation (41 TCs, 61% gap)

PORTAL tests network connectivity but NOT the settings menu UI (captive portal, SSID list, security type menu, error notifications). Add TC-NET-UI-001 for menu element validation.

5. Menu Language (5 TCs, 100% gap)

Add TC-LANG-001 covering English/Spanish/French language switching and UI string validation. Critical for multi-market certification.

6. Privacy & Legal Menus (4 TCs, 100% gap)

Add TC-PRIV-001 for Viewing Data settings, Limited Ad Tracking toggle, and Privacy & Legal menu access. Regulatory compliance (CCPA/GDPR).

Executive Insight

Of the 206 "Settings/Menu UI" test cases in the 3P manual suite, only 5 (2.6%) are fully automated by PORTAL Phase 1. Another 75 (38.3%) have partial overlap where PORTAL tests the underlying functionality (e.g., WiFi connectivity) but NOT the specific menu UI behavior (e.g., keyboard close animation, SSID list rendering).

The 116 test cases (59.2%) with zero coverage represent entire feature categories absent from PORTAL: Closed Captions, Accessibility (TalkBack/Zoom), Menu Language, Privacy settings, Bluetooth/Remote menus, and Gaming Controller menus.

Bottom line: PORTAL Phase 1 automation provides strong coverage for functional connectivity and system operations (factory reset, network connect, parental controls) but has minimal coverage of Settings menu UI behaviors — the specific area this 3P category targets.
